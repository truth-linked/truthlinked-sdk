# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-18

### Added
- Initial release of Truthlinked TypeScript/Node.js SDK
- TruthlinkedClient with full API support
- Token management (request, validate, authorize)
- Witness chain integration (submit, query, export)
- Request signing with HMAC-SHA256
- Crypto utilities (nonce generation, hashing, Ed25519 signing)
- Comprehensive TypeScript types
- Error handling with TruthlinkedError
- Express.js middleware examples
- Full documentation and examples

### Security
- Post-quantum cryptography support
- Request/response signing
- Secure nonce generation
- Timing-safe signature verification
